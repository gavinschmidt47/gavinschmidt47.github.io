Project Overview

**Artkana** is a 3-D side-scrolling momentum-based platformer featuring a unique graffiti theme and dynamic upgrade system. Developed by Slipstream Studios, the game emphasizes flow, speed, and creative expression through movement.

I was brought on as Lead Programmer specifically for my C++ proficiency demonstrated during my work on Reconnection at Mystical Die Studio. My primary focus has been refactoring the codebase to improve performance and maintainability, ensuring a solid foundation for the design team to build upon.

Gameplay Showcase

Early gameplay footage showcasing momentum mechanics and level flow.

Technical Contributions

  • System Architecture Refactoring: Converting legacy Blueprint "spaghetti code" into organized, efficient C++ structures to improve maintainability and debugging.
  • Upgrade System Overhaul: Redesigned the core upgrade mechanics to be more modular and extensible for designers.
  • Performance Optimization: Aggressively optimizing actor ticks and event loops to free up CPU resources for complex gameplay logic.
  • Designer-Centric Tools: Exposing C++ functionality to Blueprints in a clean, user-friendly way for less code-savvy team members.
